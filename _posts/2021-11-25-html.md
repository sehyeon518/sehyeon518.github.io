This is 'html'  
_In this page, let's talk about HTML_  

HTML이란 웹페이지를 만드는 데 사용하는 언어이다. HTML 문서를 웹페이지라고 하며 HTML 태그로 웹페이지의 디자인과 기능을 결정할 수 있다. 태그 사용 방법은 다음과 같다.  

```sh
<html>
    <head>                                // 시작 태그
        <title>문서의 제목입니다.</title>
    </head>                               // 종료 태그
    <body>
        <h1>제목 크기 1입니다.</h1>
        <p>이 부분은 단락입니다.</p>
    </body>
</html>
```
