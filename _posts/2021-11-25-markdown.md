This is 'markdown'  
_In this page, let's talk about Markdown_

Markdown은 텍스트 기반의 언어로 문서를 쉽고 빠르게 작성할 수 있다. HTML로 변환이 가능하며 GitHub 등 지원하는 플랫폼이 다양하다는 장점이 있다. Markdown을 사용하는 방법은 다음과 같다.

```sh
# Title      // 가장 큰 글꼴
## Title     // 두번째로 큰 글꼴
###### Title // 가장 작은 글꼴
```
# Title
## Title
###### Title

```sh
*Title* // 기울임
_Title_ // 기울임
```
*Title*
_Title_
